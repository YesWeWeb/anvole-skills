# Base OUI → Type d'équipement

Ce fichier liste les préfixes OUI (Organizationally Unique Identifier) des 3 premiers octets de l'adresse MAC, associés à un type d'équipement réseau.

**Usage :** Consulter ce fichier après avoir extrait les 3 premiers octets de la MAC.
**Note :** Un OUI inconnu ou générique (Dell, HP, Lenovo) indique un PC probable mais avec confiance réduite.

## Utilisation dans l'analyse

```
Priorité d'identification :
1. LLDP/CDP (95%) → si disponible, fait foi sur l'OUI
2. OUI dans "Phones IP" ci-dessus → phone probable (75%)
3. OUI dans "AP" → AP probable (75%)
4. OUI dans "Caméras" → caméra probable (75%)
5. OUI dans "Imprimantes" → imprimante probable (70%)
6. OUI dans "PC générique" → PC probable (60%)
7. OUI inconnu → endpoint inconnu (30%)
```

**Normalisation MAC pour lookup :**
Extraire les 3 premiers octets et comparer en majuscules sans séparateurs.
Ex: `f8:b1:56:aa:00:11` → OUI = `F8B156` → rechercher dans la table.
